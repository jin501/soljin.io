# Portfolio

Personal portfolio built with Vite + React + TypeScript + Tailwind + Framer Motion

## Stack

- **Vite** — build tool
- **React 18** + **TypeScript**
- **React Router v6** — client-side routing
- **Tailwind CSS** — utility styling
- **Framer Motion** — animations
- **Lucide React** — icons
- **Fonts:** Cormorant Garamond (display) + DM Sans (body) + DM Mono (labels)

## Getting Started

```bash
npm install
npm run dev
```

[http://localhost:5173](http://localhost:5173)

## Folder Structure

```
src/
  components/     # Nav, ProjectCard
  pages/          # HomePage, ProjectsPage, AboutPage
  hooks/          # useTheme
  data/           # All content (projects, experience, photos)
  types/          # TypeScript interfaces
  styles/         # globals.css
```
